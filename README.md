##  О проекте

**Sally Client** — это профессиональный командный клиент для обмена файлами и сообщениями, разработанный компанией MIRAJE X.

Программа позволяет:
-  Отправлять файлы другим пользователям
-  Обмениваться текстовыми сообщениями
-  Скачивать программы одной командой
-  Безопасно хранить данные с шифрованием

Клиент подключается к корпоративному серверу и обеспечивает надежный и быстрый обмен данными.

## 📥 Установка

### Вариант 1: Готовый установщик (рекомендуется)

1. Скачайте последнюю версию установщика с официального сайта MIRAJE X

2. Запустите `Sally_Client_Setup.exe`

3. Следуйте инструкциям установщика:
   - Выберите папку установки
   - При желании добавьте Sally в PATH (для запуска из любой папки)
   - Создайте ярлык на рабочем столе

4. После установки Sally доступна из любой папки через командную строку!

### Вариант 2: Ручная установка

1. Получите файл `sally.exe` у официального представителя

2. Создайте папку, например:
   ```
   C:\Sally\
   ```

3. Положите `sally.exe` в созданную папку

4. Создайте папку `datas` и положите в неё файл `logo.ico`

5. (Опционально) Добавьте папку в PATH:
   ```
   setx PATH "%PATH%;C:\Sally"
   ```

##  Быстрый старт

###  Первый запуск и регистрация

```bash
# Запустите любую команду для начала работы
sally get anything
```

При первом запуске система предложит:
1. Войти (если уже есть аккаунт)
2. Зарегистрироваться (новый пользователь)
3. Выйти

Выберите **2** и введите:
- Никнейм: `ваш_ник`
- Пароль: `********`

###  Отправка сообщения

```bash
# Отправка сообщения пользователю colleague
sally sendmes colleague "Привет, как дела?"
```

###  Отправка файла

```bash
# Отправка файла пользователю colleague
sally sendfile colleague C:\document.txt
```

###  Проверка полученных сообщений и файлов

```bash
# Открыть интерактивное меню
sally menu
```

В меню выберите:
- **1** - проверить новые сообщения
- **2** - показать полученные файлы
- **3** - показать историю сообщений

###  Скачивание программы

```bash
# Скачать программу из базы
sally get clippysave

# Скачать в конкретную папку
sally get clippysave -d D:\Downloads
```

###  Очистка

```bash
# Очистить все полученные файлы и историю
sally clearsally
```

##  Команды

### Основные команды

| Команда | Описание | Пример |
|---------|----------|--------|
| `sendmes <пользователь> <текст>` | Отправка сообщения | `sendmes colleague "Привет"` |
| `sendfile <пользователь> <файл>` | Отправка файла | `sendfile colleague file.txt` |
| `get <название>` | Скачивание программы | `get clippysave` |
| `get <название> -d <папка>` | Скачивание в папку | `get clippysave -d D:\Downloads` |
| `clearsally` | Очистка полученных файлов | `clearsally` |
| `menu` | Открыть интерактивное меню | `menu` |

### Информационные команды

| Команда | Описание |
|---------|----------|
| `-a`, `--author`, `author` | Информация об авторе |
| `-v`, `--version`, `version` | Версия программы |
| `-h`, `--help`, `help` | Справка |

### Команды в меню

| Цифра | Действие |
|-------|----------|
| `1` | Проверить новые сообщения |
| `2` | Показать полученные файлы |
| `3` | Показать историю сообщений |
| `4` | Обновить список файлов |
| `5` | Очистить все полученные файлы |
| `6` | Выйти в терминал |

##  Структура программы

После установки программа создаёт следующие файлы и папки:

```
C:\Program Files\Sally\
├── sally.exe                 # Основной исполняемый файл
├── datas/

├── received/                  # Папка для полученных файлов

└── history.json               # История сообщений
```

## ❓ Часто задаваемые вопросы

### ❔ Как зарегистрироваться?

При первом запуске любой команды программа предложит зарегистрироваться. Выберите пункт 2 и введите желаемый никнейм и пароль.

### ❔ Как войти в существующий аккаунт?

При запуске выберите пункт 1 и введите свой никнейм и пароль.

### ❔ Не работает отправка файлов?

1. Проверьте, правильно ли введён никнейм получателя
2. Убедитесь, что получатель зарегистрирован в системе
3. Проверьте интернет-соединение
4. Обратитесь в техническую поддержку

### ❔ Где хранятся полученные файлы?

Все полученные файлы сохраняются в папке `received` в директории программы:
```
C:\Program Files\Sally\received\
```

### ❔ Как очистить историю и все файлы?

```bash
sally clearsally
```

Эта команда удалит все полученные файлы и очистит историю сообщений.

### ❔ Забыл пароль. Что делать?

Обратитесь к администратору системы для сброса пароля.

### ❔ Как изменить никнейм?

Никнейм привязан к аккаунту и не может быть изменён. Для смены ника необходимо зарегистрировать новый аккаунт.

### ❔ Безопасно ли использовать программу?

Да! Все данные передаются в зашифрованном виде, пароли хешируются, а сессии шифруются с использованием современных алгоритмов.
